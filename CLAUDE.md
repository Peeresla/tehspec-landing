# Лендинг для услуг техспеца

## Что это
Одностраничный сайт-визитка. Я — техспец онлайн-школ, помогаю
с автоматизацией: GetCourse, Salebot, AmoCRM, интеграции,
Telegram-боты.

## Стек
- HTML, CSS, JavaScript (без фреймворков)
- Один файл index.html со всем стилем внутри
- Без сборщиков, без npm, без зависимостей

## Стиль
- Тёмная тема
- Современный минимализм, без лишнего
- Один акцентный цвет
- Шрифты — системные или с Google Fonts

## Правила
- Все тексты на русском
- Не выдумывай за меня кейсы и цифры — оставляй плейсхолдеры,
  если я не дал данных
- Спрашивай, если что-то непонятно

## Правила кода
- Все стили — в едином <style> внутри <head>,
  не inline в тегах
- Используй CSS-переменные для цветов в :root
- Имена классов через дефис, на английском
- Никаких external скриптов и стилей с CDN
- Не добавляй лишние строки кода, если они не относятся к запросу

## Безопасность секретов

- Никогда не коммить .env, *.key, *.pem, credentials.json
  и любые файлы с реальными токенами и паролями
- Никогда не вставляй реальные значения секретов в код,
  README, комментарии, примеры
- В примерах вместо реальных значений используй заглушки:
  BOT_TOKEN=your-token-here
- Если видишь, что где-то в коде захардкожен секрет —
  сразу скажи мне, не делая коммита
- Все секреты читать через python-dotenv из .env
