# Лендинг для услуг техспеца

## Что это
Одностраничный сайт-визитка. Я — техспец онлайн-школ, помогаю
с автоматизацией: GetCourse, Salebot, AmoCRM, интеграции,
Telegram-боты.

## Стек
- HTML, CSS, JavaScript (без фреймворков)
- Один файл index.html со всем стилем внутри
- Без сборщиков, без npm, без зависимостей

## Стиль
- Тёмная тема
- Современный минимализм, без лишнего
- Один акцентный цвет
- Шрифты — системные или с Google Fonts
